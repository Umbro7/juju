There are different ways to access and run CyberChef. Let's check the two most convenient methods!
Online Access

https://gchq.github.io/CyberChef

All you need is a web browser and an internet connection. Then, you can click this link to open CyberChef directly within your web browser.
Offline or Local Copy

https://github.com/gchq/CyberChef/releases

You can run this offline or locally on your machine by downloading the latest release file from this link. This will work on both Windows and Linux platforms. As best practice, you should download the most stable version.
